# Task 3: Email Extractor

A small Python script that finds all email addresses in a `.txt` file
and saves them into a new `.txt` file (duplicates are automatically removed).

## Key Concepts Used
- `re` — regex to match the email pattern
- `os` — check whether the input file exists
- `sys` — read command-line arguments
- File handling — reading and writing files

## How It Works
1. Reads the input `.txt` file
2. Uses a regex pattern to find all valid email addresses
3. Removes duplicates and sorts them alphabetically
4. Saves the result into a new `.txt` file

## Usage
```bash
python email_extractor.py input.txt output.txt
```

If no arguments are given, the script defaults to reading from `input.txt`
and saving results to `emails_found.txt`:
```bash
python email_extractor.py
```

## Example

**input.txt**
```
Hi, please contact sidra.dev@gmail.com for internship queries.
You can also reach out to support@codealpha.tech or hr@codealpha.tech.
Backup contact: sidra.dev@gmail.com (duplicate on purpose)
```

**output.txt**
```
hr@codealpha.tech
sidra.dev@gmail.com
support@codealpha.tech
```

## Files
- `email_extractor.py` — main script
- `input.txt` — sample input file
- `output.txt` — extracted emails (generated by the script)
