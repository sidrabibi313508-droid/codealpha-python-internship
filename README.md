# CodeAlpha Python Programming Internship

This repository contains the tasks completed as part of the **CodeAlpha Python Programming Internship**. Each task is organized in its own folder with the source code, sample files, and a dedicated README.

## Tasks

### Task 1: Hangman Game
A classic word-guessing game built in Python, where the player has a limited number of attempts to guess a hidden word letter by letter.

📁 [`Task1_HangmanGame`](./Task1_HangmanGame)

### Task 2: Stock Portfolio Tracker
A command-line tool to track a stock portfolio — add stocks, calculate total value, and view holdings.

📁 [`Task2_StockPortfolioTracker`](./Task2_StockPortfolioTracker)

### Task 3: Email Extractor
A script that scans a `.txt` file, extracts all valid email addresses using regex, and saves the unique results into a new file.

📁 [`Task3_EmailExtractor`](./Task3_EmailExtractor)

## Tech Stack
- Python 3
- Core modules: `re`, `os`, `sys`, `random`, file handling

## About
These tasks were completed as part of the CodeAlpha internship program, focusing on practical, real-world Python programming skills including automation, file handling, and building small interactive tools.
